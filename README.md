# Sector Analysis Dashboard

A real-time market intelligence dashboard for analyzing sector and stock performance.

## Features

- 📊 Real-time sector and stock analysis
- 🌓 Dark/Light mode
- 📈 Interactive charts
- 🔍 Search and filter capabilities
- 📱 Fully responsive
- 💾 Data caching for performance

## Local Development

```bash
# Install dependencies
pip install -r requirements.txt

# Run locally
python app.py
